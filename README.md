# AdsX - AI Search Advertising Agency

The agency for running ads in AI search results.

## Tech Stack

- **Framework**: Next.js 15 (App Router, TypeScript, Turbopack)
- **Styling**: Tailwind CSS
- **Fonts**: Geist Sans & Geist Mono

## Pages

- `/` - Home
- `/services` - Services
- `/case-studies` - Case Studies
- `/about` - About
- `/pricing` - Pricing
- `/contact` - Contact
- `/blog` - Blog

## Development

```bash
npm install
npm run dev
```

Visit [http://localhost:3000](http://localhost:3000)
